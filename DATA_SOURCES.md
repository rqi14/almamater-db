# Data Sources

## data/intl_aliases.json

**Contents:** Chinese names and English abbreviations for ~200 universities worldwide.

**Provenance:** Manually curated from publicly available common names. University names and abbreviations are factual designations in common use (e.g. "麻省理工学院" for MIT, "苏黎世联邦理工学院" for ETH Zurich). Sources consulted include university official websites and common usage in Chinese-language media.

**Copyright:** Individual names and abbreviations are factual and not subject to copyright. This file as compiled is an original work released under the MIT License.

## data/shuangyiliu_round1.json

**Contents:** Double First-Class (双一流) round 1 — 140 entries, 2017.

**Source:** Official MOE announcement — https://www.moe.gov.cn/jyb_xwfb/s5147/201709/t20170922_315006.html

**Notes:**
- The official announcement names 137 institutions. This file counts 140 because multi-campus institutions (e.g. 中国石油大学, 中国地质大学) are recorded as separate campus entries.
- Some discipline names may reflect aggregated labels rather than exact official MOE 一级学科 names.
